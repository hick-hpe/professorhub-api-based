## Cadastrar Calendário Letivo
| UC04 | Descrição |
|------|-----------|
| Nome | Cadastrar Calendário Letivo |
| Descrição Geral | Permite que o docente cadastre calendários para disciplinas |
| Atores | Docente |
| Entrada | Identificador, data de início, data de fim |
| Pré-condições | O docente deve estar logado no sistema |
| Fluxo Principal | <ul><li>Docente acessa cadastro de calendário</li><li>Docente informa os dados do calendário</li><li>Docente confirma cadastro</li><li>Sistema valida dados</li><li>Sistema registra o calendário</li></ul> |
| Fluxos Alternativos | <ul><li>Datas inválidas: sistema solicita correção</li></ul> |
| Pós-condições | Calendário cadastrado no sistema |
| Prioridade | Alta |
