## Editar dados do Plano de Aula
| UC06 | Descrição |
|------|-----------|
| Nome | Editar dados do Plano de Aula |
| Descrição Geral | Permite que o docente edite os dados de um plano de aula |
| Atores | Docente |
| Entrada | ID do plano de aula, dados a serem alterados |
| Pré-condições | O docente deve estar logado no sistema |
| Pós-condições | Dados do plano de aula atualizados |
| Prioridade | Alta |

## Adicionar Aulas
| UC08 | Descrição |
|------|-----------|
| Nome | Adicionar Aulas |
| Descrição Geral | Permite adicionar aulas manualmente |
| Atores | Docente |
| Entrada | Título, quantidade de aulas |
| Pré-condições | O docente deve estar logado no sistema |
| Fluxo Principal | <ul><li>Docente acessa disciplina</li><li>Docente seleciona adicionar aula</li><li>Docente informa dados da aula</li><li>Sistema registra aula</li></ul> |
| Fluxos Alternativos | <ul><li>Dados inválidos: sistema solicita correção</li></ul> |
| Pós-condições | Aula cadastrada no sistema |
| Prioridade | Alta |
