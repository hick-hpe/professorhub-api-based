# Casos de Uso do Sistema

Este documento descreve os principais casos de uso do sistema, detalhando o fluxo principal de execução e possíveis fluxos alternativos.

## Casos de uso
- [Usuários](./casos/usuarios.md): gerenciamento das contas, incluindo cadastro, autenticação (login/logout) e recuperação;
- [Disciplinas](./casos/disciplinas.md): cadastro, edição e exclusão de disciplinas, além da definição de informações como carga horária, dias de aula, objetivos e ementa;
- [Calendários](./casos/calendarios.md): criação e gerenciamento do calendário letivo das disciplinas, incluindo definição de datas e organização das aulas;
- [Tarefas](./casos/tarefas.md): cadastro, edição e exclusão de tarefas associadas às disciplinas, permitindo o acompanhamento de prazos e atividades;
- [Avaliações](./casos/avaliacoes.md): cadastro, edição e exclusão de avaliações, permitindo ao docente organizar prazos e tipos de avaliação das disciplinas;
