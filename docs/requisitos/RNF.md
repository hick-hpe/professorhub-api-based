# Requisitos Não Funcionais

| ID   | Requisito                                                                 | Prioridade |
|------|---------------------------------------------------------------------------|------------|
| RNF01 | O projeto deve ser desenvolvido utilizando o framework Django.           | Alta       |
| RNF02 | Os dados devem ser armazenados em um banco de dados relacional PostgreSQL. | Alta       |
| RNF03 | O sistema deve exigir autenticação para acessar funcionalidades de planejamento. | Alta |
| RNF04 | O sistema deve garantir integridade e consistência no agendamento das aulas. | Alta |
| RNF05 | O código-fonte deve ser versionado utilizando Git e hospedado no GitHub. | Alta |
| RNF06 | A aplicação deve ser responsiva e acessível.                              | Média      |
| RNF07 | A interface deve ser intuitiva e de fácil navegação para docentes.        | Média      |
