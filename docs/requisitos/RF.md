# Requisitos Funcionais

### Conta
| ID   | Requisito                                               | Prioridade |
|------|---------------------------------------------------------|------------|
| RF01 | O sistema deve permitir o cadastro de docentes          | Alta       |
| RF02 | O sistema deve permitir o login de docentes             | Alta       |
| RF03 | O sistema deve permitir a recuperação de conta          | Média      |
| RF04 | O sistema deve permitir a redefinição da senha          | Média      |
| RF05 | O sistema deve permitir a alteração dos dados do perfil | Média      |
| RF06 | O sistema deve permitir a exclusão da conta             | Baixa      |

### Disciplina
| ID   | Requisito                                                                                   | Prioridade |
|------|---------------------------------------------------------------------------------------------|------------|
| RF07 | O sistema deve permitir que o docente cadastre disciplinas                                 | Alta       |
| RF08 | O sistema deve permitir editar os dados de uma disciplina                                  | Média      |
| RF09 | O sistema deve permitir excluir uma disciplina                                             | Média      |
| RF10 | O sistema deve permitir que o docente planeje aulas vinculando conteúdos, atividades e datas | Alta       |
| RF11 | O sistema deve permitir visualizar dados sobre carga horária e conteúdos programáticos     | Média      |

### Plano de Aula
| ID   | Requisito                                                                                   | Prioridade |
|------|---------------------------------------------------------------------------------------------|------------|
| RF12 | O sistema deve permitir criar um plano de aula                                             | Alta       |
| RF13 | O sistema deve permitir visualizar os planos de aula cadastrados                          | Alta       |
| RF14 | O sistema deve permitir editar um plano de aula existente                                  | Média      |
| RF15 | O sistema deve permitir excluir um plano de aula                                           | Média      |
| RF16 | O sistema deve atualizar as datas dos planos de aula caso um plano tenha seu dia alterado ou seja excluído | Média |
| RF17 | O sistema deve permitir duplicar um plano de aula                                          | Baixa      |

### Avaliação
| ID   | Requisito                                                         | Prioridade |

| RF22 | O sistema deve permitir vincular uma avaliação a um plano de aula | Alta       |

|------|--------------------------------------------------------------------|------------|

| RF28 | O sistema deve permitir alterar o período letivo                  | Média      |

| RF30 | O sistema deve permitir excluir um calendário                     | Média      |
